# Twitter-Like

*ASP.NET MVC Twitter-Like Web Application with Entity Framework*

**Features:**  
  * Authentication (provided by ASP.NET Identity)  
  * Create, Like, Retweet, Reply and Report a Tweet.  
  * Search for Users.  
  * Follow Users.  
  
**Javascript and CSS libraries:**  
  * jQuery  
  * animate.css  
  * bootbox.js
